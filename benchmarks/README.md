# Agent benchmarks

Community-maintained harness for measuring how well coding agents write [Zero](https://zerolang.ai) programs versus Python:

**[github.com/HKTITAN/zero-agent-bench](https://github.com/HKTITAN/zero-agent-bench)**

- 15 CLI tasks, structured fix loop (`zero check --json` feedback)
- Models: Claude Opus 4.7, Sonnet 4.6, Haiku 4.5
- Published results: [results/RESULTS.md](https://github.com/HKTITAN/zero-agent-bench/blob/main/results/RESULTS.md)

Pattern skills that improve agent pass rates: **[github.com/HKTITAN/zero-skills](https://github.com/HKTITAN/zero-skills)**
